# Standard

各種ツールや構造体で使用する標準ライブラリ。

## 動作確認

`3ds Max 2022.3 Update`

## インストール

`install.ms`を実行する。

## アンインストール

`uninstall.ms`を実行する。

## スタンドアローン版

### インストール

`Distribution\Standard.min.ms`を実行する。

### アンインストール

```maxscript
::std.Uninstall()
```
